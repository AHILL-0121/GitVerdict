# Static Assets

This directory contains public static files served by Next.js.

## Files

- **`favicon.svg`** — GitVerdict icon (BLAZE mascot)
- **`blaze-mascot.svg`** — Reference SVG for BLAZE mascot
- **`blaze-states-showcase.html`** — Interactive showcase of all 10+ BLAZE states

## Usage

Files in this directory are served from the root URL path.

Example:
- `public/favicon.svg` → accessible at `/favicon.svg`
- `public/og-image.png` → accessible at `/og-image.png`

## Reference Files

The `blaze-mascot.svg` and `blaze-states-showcase.html` files are reference/development assets showing the mascot design system. They are not used in the production app but kept for documentation purposes.
